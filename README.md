# SQL-Learning 📚

Welcome to my SQL Learning repository! This space is dedicated to documenting my day-by-day journey of mastering SQL (Structured Query Language). 

## 🎯 About This Repository

As I learn new SQL concepts, I am organizing my notes, syntax examples, and key takeaways into structured, easy-to-read Markdown files. The goal is to build a comprehensive, sequential guide that serves as both a personal reference and a helpful resource for anyone else learning SQL.

## 📂 Repository Structure

The files in this repository are organized in a logical learning sequence. Currently covering fundamental concepts, the structure will evolve as I dive deeper into advanced topics, queries, and database management. 

*(More files will be added as my learning progresses!)*

* `01_SQL_Fundamentals.md` - Introduction to databases, DBMS, and basic SQL concepts.
* *(Coming soon: SQL Syntax, Data Types, Joins, Subqueries, and more)*

## 🚀 What's Inside?

For each topic, you will typically find:
* **Theory & Concepts:** Clear explanations of what things are and why they matter.
* **Key Points:** Important rules and best practices.
* **Syntax & Examples:** Practical snippets showing how to write the code.
